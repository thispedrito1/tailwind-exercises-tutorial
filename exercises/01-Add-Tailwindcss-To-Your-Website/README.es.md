# `01` Add Tailwind CSS To Your Website

Hay dos formas de agregar Tailwind CSS a tu sitio web: **Remoto** o **Local**. En este ejercicio usaremos la versión remota por CDN.

+ Así es como se realiza una importación **remota** de Tailwind CSS:

```html
<script src="https://cdn.tailwindcss.com"></script>
```

### 🔎 Importante:

El script CDN de Tailwind debe colocarse dentro de la etiqueta `<head>`.

También puedes agregar Tailwind desde una configuración **local** usando npm y un proceso de compilación (uso avanzado).

### 🔎 Importante:

> Para más información: https://tailwindcss.com/docs/installation/play-cdn


## 📝 Instrucciones:

1. Agrega este script a tu archivo para importar Tailwind CSS con éxito:

```html
<script src="https://cdn.tailwindcss.com"></script>
```

2. Mantén el botón existente y aplícale clases de Tailwind para que se vea rojo.

## 💡 Pista:

+ Usa clases como `bg-red-500`, `text-white` y `px-4 py-2`.
