# `01` Add Tailwind CSS To Your Website

There are two ways to add Tailwind CSS into your website:
**Remote** or **Local**. In this exercise, we will use the remote CDN version.

+ This is how you do a **remote** Tailwind CSS import:

```html
<script src="https://cdn.tailwindcss.com"></script>
```

### 🔎 Important:

The Tailwind CDN script should be placed inside the `<head>` tag.

+ You can also add Tailwind from a **local** setup using npm and a build step (advanced usage).

> For more information: https://tailwindcss.com/docs/installation/play-cdn

## 📝 Instructions:

1. Add the CDN script in your file to import Tailwind CSS successfully:

```html
<script src="https://cdn.tailwindcss.com"></script>
```

2. Keep the existing button and style it with Tailwind utility classes so it looks red.

## 💡 Hint:

+ Use classes like `bg-red-500`, `text-white`, and `px-4 py-2`.
